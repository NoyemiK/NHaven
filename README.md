## NHaven

An exploratory fork of Haven in order to add custom features and change
existing things in the codebase. see the original project [here](https://github.com/ancsemi/Haven)

It isn't meant for general use, so I don't intend to answer/fix any bugs
brought up in the issue tracker.

### Disclaimer:

**I am not a web developer!** I know nothing about the domain object model and
I barely even know what a JS even is. That said, my hacks at this codebase are
all my own with the help of JS documentation and headaches I incurred poking at
the code. No LLM usage was or ever will be used in my commits. The original
lists claude code as a contributor, which is part of (but not the entirety of)
the reason I decided to fork it. No offense is meant to the original developer,
I know he's doing his best.

Many commits will show large diffs. The reason for this is because I change the
formatting to `tab = 4spaces` because the original had `tabs = 2spaces` and
this made it hard for me to read.
